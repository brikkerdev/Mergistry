

# MERGISTRY — Game Design Document

---

## 1. Концепция

### Elevator Pitch

Пошаговый roguelike, где ты алхимик, спускающийся в подземелье. Твоё оружие — зелья, которые ты варишь на доске перегонки между боями. Каждый ран — 8–12 минут. Полностью офлайн.

### Платформа

Мобильные устройства (Android — приоритет, затем iOS). Портретная ориентация. Минимальное разрешение — 720×1280. Целевое — 1080×1920.

### Технологии

Unity 6.3, Built-in Render Pipeline 2D. Без растровых текстур — весь визуал на шейдерах (SDF-примитивы, процедурный noise, glow, bloom). Спрайты допускаются только для иконок UI.

### Целевая аудитория

Казуальные и мидкор игроки 14–35 лет. Любители таймкиллеров, знакомые с merge-играми или roguelike (Slay the Spire, Threes, 2048).

### Монетизация

Вне скоупа этого документа. Игра проектируется как полноценный продукт без pay-to-win.

### Ключевые столпы дизайна

| Столп | Значение |
|-------|----------|
| **Быстрые решения** | Каждое действие — осознанный выбор с последствиями. Нет пустых ходов |
| **Читаемость** | Всё видно и понятно на 5-дюймовом экране за 0.5 секунды |
| **Emergent depth** | Простые правила, сложные комбинации. Глубина из взаимодействия, а не из количества |
| **Juice** | Каждое действие чувствуется. Визуальная и тактильная отдача |

## 2. Игровой цикл

### Структура сессии

```
Лаборатория (меню) → Ран → Результат → Лаборатория

Один ран:
  Этаж 1 → Этаж 2 → Этаж 3 → Победа или смерть

Один этаж:
  [Перегонка] → [Бой] → [Перегонка] → [Бой] → ... → [Элита/Босс]

Один бой:
  4–7 пошаговых ходов на сетке 5×5
```

### Длительность

| Элемент | Время |
|---------|-------|
| Фаза перегонки | 15–30 сек |
| Один бой | 30–60 сек |
| Один этаж | 3–4 мин |
| Полный ран | 8–12 мин |

### Петля вовлечения

```
Перегонка: "Что я могу сварить из этой доски?"
    ↓
Бой: "Как применить то, что сварил?"
    ↓
Карта: "Какой путь выбрать дальше?"
    ↓
Перегонка: "Мне нужно зелье против следующего врага"
    ↓
... (повторять до победы или смерти)
```

## 3. Перегонка (merge-система)

### Общее описание

Между каждым боем игрок попадает на экран перегонки. Это сетка 4×4 (16 ячеек), заполненная **ингредиентами** — каплями пяти базовых элементов. Игрок комбинирует ингредиенты, создавая **варева** (промежуточные объекты), которые в конце фазы собираются в инвентарь зелий.

Перегонка — это пространственная головоломка. Каждая доска уникальна, и оптимальный ход зависит от расположения ингредиентов, текущего инвентаря зелий и предстоящего боя.

### Объекты на доске

На доске существует ровно два типа объектов:

| | Ингредиент | Варево |
|---|---|---|
| **Что это** | Капля базового элемента | Зелье в процессе создания |
| **Визуал** | Цветная капля (SDF metaball, покачивается) | Светящаяся колба с жидкостью внутри |
| **Уровень** | Нет — ингредиенты всегда базовые | lv1, lv2 или lv3 |
| **Может быть перетянут** | Да | Нет (неподвижно) |
| **Цель перетягивания** | На соседний ингредиент (merge) или на соседнее варево (подпитка) | — |

### Действия

За одну фазу перегонки — **3 действия** (визуализированы тремя светящимися точками над доской). Каждое действие = одно перетягивание (drag) по горизонтали или вертикали на соседнюю клетку. Диагональное перетягивание запрещено.

### Два типа действий

#### Действие 1: Merge (ингредиент → ингредиент)

Перетащить ингредиент на **соседний ингредиент**. Результат:

- Оба ингредиента потребляются
- В клетке **назначения** (куда тащили) появляется **варево lv1**
- Клетка **источника** (откуда тащили) становится **пустой**
- Тип варева определяется рецептом (два одинаковых → базовое зелье, два разных → рецептурное)

```
До:     🔴 🔵 🟢 🟡        После:    __ [💨¹] 🟢 🟡
        🟢 🔴 🟡 🔵                  🟢  🔴  🟡 🔵
        
        Drag 🔴(0,0) → 🔵(1,0)
        Merge: 🔴+🔵 = Пар lv1
        Клетка (0,0) пуста, клетка (1,0) = варево Пар lv1
```

#### Действие 2: Подпитка (ингредиент → варево)

Перетащить ингредиент на **соседнее варево**. Результат:

- Ингредиент потребляется (клетка пуста)
- Варево повышается на **+1 уровень** (lv1→lv2 или lv2→lv3)
- Ингредиент **должен совпадать** с одним из элементов рецепта варева

Допустимые ингредиенты для подпитки:

| Варево | Допустимые ингредиенты |
|--------|----------------------|
| Пламя (🔴+🔴) | Только 🔴 |
| Поток (🔵+🔵) | Только 🔵 |
| Отрава (🟢+🟢) | Только 🟢 |
| Сияние (🟡+🟡) | Только 🟡 |
| Мрак (🟣+🟣) | Только 🟣 |
| Пар (🔴+🔵) | 🔴 **или** 🔵 |
| Напалм (🔴+🟢) | 🔴 **или** 🟢 |
| Кислота (🔵+🟢) | 🔵 **или** 🟢 |
| ... (все рецептурные) | Любой из двух составляющих |

```
До:     __ [💨¹] 🟢 🟡        После:    __ [💨²] 🟢 🟡
        🟢  🔴  🟡 🔵                  🟢  __  🟡 🔵
        
        Drag 🔴(1,1) → [💨](1,0)
        Подпитка: 🔴 входит в рецепт Пара (🔴+🔵) ✓
        Пар lv1 → Пар lv2
```

#### Недопустимая подпитка

Если ингредиент **не входит** в рецепт варева — перетягивание запрещено. Клетка варева не подсвечивается как допустимая цель, drag не срабатывает. Действие не тратится.

#### Максимальный уровень

Варево lv3 — максимум. Попытка подпитать lv3 варево запрещена (аналогично недопустимой подпитке).

### Формула уровней

```
1 действие  (merge: 2 ингредиента)           → lv1
2 действия  (merge + 1 подпитка: 3 ингредиента) → lv2
3 действия  (merge + 2 подпитки: 4 ингредиента) → lv3
```

### Ключевой trade-off

Три действия за фазу. Как потратить:

| Стратегия | Результат | Когда выгодно |
|-----------|-----------|---------------|
| 3 × merge | 3 зелья lv1 | Нужно разнообразие, пустой инвентарь |
| 1 merge + 1 подпитка + 1 merge | 1 зелье lv2 + 1 зелье lv1 | Баланс силы и разнообразия |
| 1 merge + 2 подпитки | 1 зелье lv3 | Нужна максимальная мощь, инвентарь полон |

Решение зависит от:
- Состояния инвентаря (есть ли свободные слоты?)
- Предстоящего боя (видны иконки врагов)
- Текущего HP (рисковать ли?)
- Расклада доски (есть ли цепочки для подпитки?)

### Конец фазы

Фаза завершается когда:
- Потрачены все 3 действия, **или**
- Игрок нажимает кнопку «В бой →»

При завершении:
1. Все варева на доске **собираются в инвентарь** (анимация: колбы взлетают и встают в слоты)
2. Оставшиеся ингредиенты **растворяются** (анимация: fade out с частицами)
3. Пустая доска исчезает
4. Начинается бой

### Генерация доски

Каждая фаза — полностью новая доска. Алгоритм генерации:

1. Заполнить 16 ячеек случайными элементами из доступного пула
2. Распределение: примерно равные доли каждого элемента (±1)
3. **Гарантия**: минимум 3 пары соседних ингредиентов, дающих валидный merge
4. На Этаже 1 — только 3 элемента (🔴, 🔵, 🟢). На Этаже 2 — добавляется 🟡. На Этаже 3 — все 5

Пул элементов на доске контролируется **разблокированными рецептами**: если рецепт не открыт, элементы для него всё равно могут быть на доске (они нужны для базовых зелий), но merge разных неизвестных элементов запрещён.

### Пространственная глубина

Почему позиция на доске важна:

**Направление merge определяет соседей варева.** Merge 🔴→🔵 вправо оставляет варево справа. У него одни соседи. Merge 🔵←🔴 влево оставляет варево слева. У него другие соседи. Выбор направления — это выбор будущих возможностей подпитки.

**Пустые клетки ломают цепочки.** Потребление ингредиентов оставляет пустоты. Пустая клетка — это разрыв: нельзя перетягивать через неё. Порядок действий критичен.

**Базовые зелья сложнее прокачать.** Пламя (🔴+🔴) требует для подпитки только 🔴. Четыре 🔴 в цепочке — редкость. Пар (🔴+🔵) принимает и 🔴, и 🔵 — проще найти подходящий соседний ингредиент. Это **встроенный стимул** изучать рецепты.

### Полный пример фазы

```
Доска в начале фазы:
🔴 🔵 🔴 🟢
🟢 🔴 🟡 🔵
🔵 🟢 🔵 🟡
🟡 🔵 🟢 🔴

Цель игрока: Пламя lv2 + Кислота lv1

Действие 1 [Merge]: Drag 🔴(0,0) → 🔴(0,1)

  __  🔵 🔴 🟢
  [🔥¹] 🔴 🟡 🔵       ← варево Пламя lv1 в (0,1)
  🔵  🟢 🔵 🟡
  🟡  🔵 🟢 🔴

  Соседи варева [🔥¹](0,1): ↑🔵(0,0)пусто, →🔴(1,1), ↓🔵(0,2)
  🔴(1,1) подходит для подпитки!

Действие 2 [Подпитка]: Drag 🔴(1,1) → [🔥¹](0,1)

  __  🔵 🔴 🟢
  [🔥²] __ 🟡 🔵       ← Пламя повысилось до lv2
  🔵  🟢 🔵 🟡
  🟡  🔵 🟢 🔴

Действие 3 [Merge]: Drag 🔵(0,2) → 🟢(1,2)

  __  🔵 🔴 🟢
  [🔥²] __ 🟡 🔵
  __  [🧪¹] 🔵 🟡     ← варево Кислота lv1 в (1,2)
  🟡  🔵  🟢 🔴

Конец фазы:
→ Пламя lv2 и Кислота lv1 собираются в инвентарь
→ Оставшиеся ингредиенты растворяются
→ Начинается бой
```

| Параметр | Lv1 | Lv2 | Lv3 |

## 5. Инвентарь зелий

### Слоты

4 слота (расширяется до 5 через мета-прогрессию).

### Жизненный цикл зелья

1. **Создание**: варево собирается с доски перегонки → попадает в свободный слот
2. **Обновление**: если зелье того же типа уже есть в инвентаре — сохраняется **более высокий** уровень. Слот не дублируется
3. **Заполненный инвентарь**: если все слоты заняты и зелье нового типа — игрок **выбирает слот для замены** (выбрасываемое зелье растворяется с анимацией)
4. **Использование в бою**: зелья **не тратятся** при броске. Они доступны весь ран. Ограничение — кулдаун
5. **Смена**: зелье можно заменить только через перегонку, события или реликвии

### Кулдаун

После использования зелье уходит в кулдаун:
- **lv1 и lv2**: 2 хода
- **lv3**: 1 ход

Визуализация: зелье тускнеет, поверх — число оставшихся ходов до готовности. Каждый ход (включая ход врагов) уменьшает счётчик на 1.

### Отображение

```
┌────┐ ┌────┐ ┌────┐ ┌────┐
│ 🔥 │ │⚡🔒│ │ 🧪 │ │    │
│ ▪▪ │ │ 1  │ │ ▪  │ │    │
└────┘ └────┘ └────┘ └────┘
ready   cd:1   ready  empty

▪ = точки уровня (▪ = lv1, ▪▪ = lv2, ▪▪▪ = lv3)
🔒 + число = ходов до готовности
```

Каждый слот визуально — SDF-колба. Внутри — анимированная жидкость цвета зелья. При кулдауне жидкость темнеет, колба покрывается «инеем» (noise overlay). При готовности — пульсирующее свечение.

## 6. Боевая система

### Сетка

- Обычный бой: **5×5** (25 клеток)
- Босс-бой: **6×6** (36 клеток)

Клетки — квадратные, равномерно заполняют доступную часть экрана (~70% высоты). Каждая клетка имеет рамку, видимую при подсветке. В спокойном состоянии — только тонкие разделители (Voronoi noise шейдер имитирует каменный пол).

### Ход игрока

Каждый ход состоит из двух частей. Порядок — **любой** (сначала двигайся, потом кидай, или наоборот):

**1. Движение** — от 0 до 2 клеток (по горизонтали, вертикали, или комбинация — L-образное движение допускается). Нельзя проходить через врагов и непроходимые клетки.

**2. Действие** — одно из:
- **Бросить зелье**: выбрать зелье из инвентаря → выбрать целевую клетку в пределах дальности → зелье применяется
- **Толкнуть врага**: враг должен быть на соседней клетке → свайп в направлении толчка → враг сдвигается на 1 клетку в этом направлении (если клетка свободна). Если за врагом стена — враг получает 1 доп. урон. Если за врагом яма (модификатор) — мгновенная смерть
- **Пропустить**: восстановить 1 HP (не выше максимума)

Обе части обязательны, но можно выбрать «0 движения» и/или «пропуск действия».

### Ход врагов

После хода игрока все враги ходят **одновременно**. Каждый враг выполняет своё намерение (отображённое до хода игрока). Порядок разрешения:

1. Все передвижения врагов
2. Все атаки врагов
3. Все эффекты зон (DoT, замедление и т.д.)
4. Уменьшение таймеров (кулдауны зелий, длительности зон, длительности эффектов)

### Индикация намерений

Перед ходом игрока каждый враг отображает:

| Индикатор | Что показывает | Визуал |
|-----------|---------------|--------|
| Иконка действия | Тип: ⚔️ атака, 🛡 защита, 📢 призыв, 🚶 движение | Над головой врага |
| Зона действия | Клетки, куда направлена атака | Красная подсветка клеток (пульсирующая) |
| Число урона | Сколько урона нанесёт | Число над иконкой |
| Направление движения | Куда враг пойдёт | Стрелка на клетке назначения |

Игрок **всегда** знает что произойдёт на следующем ходу врагов. Бой — это шахматы, не рулетка.

### Зоны на поле

Некоторые зелья оставляют зоны на клетках после применения:

| Зона | Создаётся | Длительность (lv1) | Эффект |
|------|-----------|---------------------|--------|
| 🔥 Огонь | Пламя, Напалм | 2 хода | 1 урон при входе и в начале хода |
| 💧 Вода | Поток | 3 хода | Замедление (движение -1 клетка) |
| ☠️ Яд | Отрава, Напалм | 3 хода | 1 урон в начале каждого хода |
| ❄️ Лёд | — (только через combo) | 2 хода | Скольжение (существо проезжает на 1 клетку дальше) |

Зоны влияют и на игрока, и на врагов. Игрок получает урон от своих зон, если встанет на них.

### Элементальные combo

Бросок зелья **в клетку с существующей зоной** вызывает усиленный эффект:

| Зелье | + Зона | Combo-эффект |
|-------|--------|-------------|
| ⚡ Молния | 💧 Вода | ×2 урон **всем** стоящим в любой клетке воды на поле |
| 🔥 Пламя | ☠️ Яд | Ядовитое облако расширяется: AoE яда +1 клетка во все стороны |
| 🌊 Поток | ❄️ Лёд | Заморозка: все враги на воде пропускают 2 хода |
| 🔥 Пламя | ❄️ Лёд | Пар: ослепление всех в зоне на 2 хода |
| ⚡ Молния | 🔥 Огонь | Огненная буря: 3×3 AoE, повышенный урон |
| 🧪 Кислота | 💧 Вода | Кислотный потоп: все клетки воды наносят 2 урона |
| ☠️ Отрава | 🔥 Огонь | Токсичный дым: 3×3 облако, DoT 3 хода + ослепление |

Combo — **главный источник дофамина**. При срабатывании:
- Усиленный screenshake
- Slowmo 0.15 сек
- Надпись «COMBO!» с glow
- Звуковой акцент
- Расширенные визуальные эффекты

### Дальность броска зелий

Все зелья имеют дальность броска (от позиции игрока):

| Тип | Дальность |
|-----|-----------|
| Облако, Крест, Область | 3 клетки |
| Линия | Начинается от игрока, идёт в выбранном направлении |
| Одиночная | Указана в описании зелья (3–5 клеток) |

### Управление (мобильное)

| Действие | Жест |
|----------|------|
| Движение | Свайп **от персонажа** в направлении. Призрак (полупрозрачный силуэт) показывает конечную позицию до отпускания пальца. Отпустить палец = подтвердить |
| Бросок зелья | **Тап** на зелье внизу → подсвечиваются доступные клетки (синий) → **тап** на целевую клетку |
| Толчок | **Свайп** от врага на соседней клетке в направлении толчка (стрелка показывает направление до отпускания) |
| Пропуск хода | Кнопка ⏭️ или **тап** на своего персонажа |
| Отмена | **Тап** на пустое место или кнопка ↩️ (отменяет текущий выбор зелья) |

### Undo

Движение и действие можно отменить **до подтверждения хода** (кнопка ✓). Кнопка ↩️ в углу — полный undo текущего хода.

## 7. Враги

### Принципы дизайна врагов

- Каждый враг = **уникальная механическая задача**
- Поведение **полностью предсказуемо** (показаны намерения)
- Визуально читается за 0.5 секунды на маленьком экране
- Реализован целиком SDF-шейдерами (без спрайтов)
- Цветовая индикация: каждый тип врага имеет уникальный акцентный цвет

## 8. Боссы

### Принципы

- Сетка **6×6**
- 2 фазы (поведение меняется при 50% HP)
- 8–12 ходов на бой
- Уникальная механика, **не встречавшаяся** у обычных врагов
- Перед боссом — **всегда** фаза перегонки
- Занимает **2×2 клетки** на сетке (визуально крупнее обычных врагов)

### Босс Этажа 1: Королева пауков

| Параметр | Значение |
|----------|----------|
| HP | 10 |
| Размер | 2×2 |
| Сетка | 6×6. Стартовые 3-4 клетки с паутиной (замедление) |

**Фаза 1 (HP 10–5):**
- Каждый ход стреляет паутиной в **2 линии** (вертикаль + горизонталь от своей позиции)
- Двигается к игроку на 1 клетку
- Паутина на клетках = замедление (движение игрока -1)

**Фаза 2 (HP 5–0):**
- Призывает 2 пауков (обычных)
- Отступает к краю сетки
- Стреляет паутиной в 3 линии

**Контрплей:** Выжигать паутину огнём (Пламя, Напалм). Пробираться к ней за заслон пауков. Combo: Молния + паутина (вода) = усиленный урон.

---

### Босс Этажа 2: Железный голем

| Параметр | Значение |
|----------|----------|
| HP | 14 (6 брони + 8 HP) |
| Размер | 2×2 |
| Сетка | 6×6. 2 каменных столба (непроходимые клетки 1×1) |

**Фаза 1 (HP 14–7):**
- Каждый ход бьёт по области 2×2 перед собой
- Притягивает игрока на 1 клетку (как магнит-голем)
- Двигается на 1 клетку к игроку

**Фаза 2 (HP 7–0):**
- Ломает столбы (AoE 2×2 осколки в зоне каждого столба, 3 урона)
- Ускоряется: двигается на **2** клетки за ход
- Перестаёт притягивать

**Контрплей:** Кислота для брони. Столбы как укрытие в фазе 1 (блокируют притяжение и удар). В фазе 2 — маневрировать и использовать зоны.

---

### Босс Этажа 3: Алхимик-Отступник

| Параметр | Значение |
|----------|----------|
| HP | 16 |
| Размер | 1×1 (человеческий размер) |
| Сетка | 6×6. 3 котла (интерактивные объекты, 1×1, HP 3 каждый) |

**Фаза 1 (HP 16–8):**
- Копирует зелья игрока (как зеркальный слайм, но с уровнем на 1 ниже)
- Каждый ход создаёт зону (случайная из: огонь, яд, вода) в случайной клетке
- Двигается к ближайшему котлу

**Фаза 2 (HP 8–0):**
- Котлы начинают **каждый ход** генерировать случайные зоны вокруг себя (2×2)
- Босс телепортируется **между котлами** (каждый ход — к следующему)
- Если все котлы уничтожены — телепортация прекращается, босс паникует (урон ×0.5, но стоит на месте)

**Контрплей:** Уничтожить котлы (3 HP каждый) — лишить босса телепортации. Использовать его зоны для combo. В фазе 2 — предсказать к какому котлу он прыгнет и подготовить зону для combo.

## 9. Структура рана

### Карта этажа

Между боями игрок видит карту текущего этажа — вертикальную последовательность узлов с развилками. Карта генерируется при входе на этаж.

```
        [👑 БОСС]
        /       \
    [⚔️ Бой]  [❓ Событие]
      |           |
  [⭐ Элита]  [⚔️ Бой]
    /       \     |
[⚔️ Бой] [❓ Событие]
    \       /
    [🧙 СТАРТ]
```

Правила генерации карты:
- 3–4 узла от старта до босса
- Всегда ровно **1 элита** на этаж
- 1–2 события на этаж
- Минимум 2 пути (развилка)
- Узлы одного ряда могут быть связаны (множественные входы)
- Босс — всегда один, все пути ведут к нему

Игрок видит **иконки** узлов, но не содержимое боёв (состав врагов скрыт). Тип узла (бой / элита / событие / босс) всегда виден.

### Типы узлов

| Узел | Иконка | Что происходит | Награда |
|------|--------|---------------|---------|
| ⚔️ Бой | Скрещённые мечи | Перегонка → бой с 2–3 врагами | — |
| ⭐ Элита | Звезда | Перегонка → бой с усиленным врагом | Выбор реликвии (1 из 3) |
| ❓ Событие | Вопросительный знак | Случайное событие (см. ниже) | Зависит от события |
| 👑 Босс | Корона | Перегонка → босс-бой | Переход на следующий этаж |

### Перегонка перед каждым узлом

Перед **каждым** боевым узлом (бой, элита, босс) — фаза перегонки. Перед событиями перегонки нет.

### События

---

**⛲ Фонтан**

Экран: анимированный фонтан (SDF + particle-like noise). Два варианта:

| Выбор | Эффект |
|-------|--------|
| [Выпить] | 70% — полное лечение HP. 30% — потеря 1 max HP (необратимо на ран) |
| [Наполнить колбу] | Получить случайное зелье lv2 в свободный слот. Если нет слота — выбрать замену |

---

**🛒 Торговец**

Экран: три зелья на прилавке. Каждое имеет цену в HP.

| Параметр | Значение |
|----------|----------|
| Ассортимент | 3 случайных зелья (lv1–lv2, из разблокированных рецептов) |
| Цена | lv1 = 1 HP, lv2 = 2 HP |
| Ограничение | Можно купить только 1 зелье за визит |
| Отказ | Можно уйти без покупки |

---

**⚗️ Алтарь**

Экран: каменный алтарь с рунами (SDF + glow). Предложение:

| Выбор | Эффект |
|-------|--------|
| [Пожертвовать зелье] | Удалить 1 зелье из инвентаря → получить случайную реликвию из разблокированного пула |
| [Уйти] | Ничего не происходит |

---

**🎁 Сундук**

Экран: сундук открывается (SDF + анимация). Выбор 1 из 3:

| Вариант | Эффект |
|---------|--------|
| 🧪 Зелье | Случайное зелье lv1 (из разблокированных) |
| ❤️ Лечение | +3 HP (не выше максимума) |
| 🔮 Бонус | +1 действие на следующей фазе перегонки (4 действия вместо 3) |

### Модификаторы комнат

Каждая боевая комната на Этаже 2+ **может** иметь модификатор (50% шанс, случайный из пула). Модификатор отображается иконкой рядом с узлом на карте.

| Модификатор | Иконка | Эффект на сетку |
|-------------|--------|-----------------|
| 🌊 Затопленная | Волна | Вся сетка = зона воды. Молния ×2 урон, огонь ×0.5 урон |
| 🔥 Горящая | Огонь | Каждый ход 1–2 случайные пустые клетки загораются (зона огня, 2 хода) |
| 🕳 С ямами | Дыра | 2–3 клетки = ямы (непроходимые). Толчок врага в яму = мгновенное убийство |
| ⏳ Таймер | Часы | Через 6 ходов потолок рухнет: все живые (включая игрока) получают 99 урона. Убей всех или умри |
| 🌑 Тёмная | Глаз | Игрок видит только **2 клетки** вокруг себя. Остальное — туман (но намерения видимых врагов показаны) |
| 🔄 Конвейер | Стрелки | Все существа (и игрок!) сдвигаются на 1 клетку в одном направлении в конце каждого хода. Направление показано стрелками на полу |

## 10. Характеристики игрока

### Здоровье

| Параметр | Значение |
|----------|----------|
| Стартовое HP | 5 |
| Максимум HP | 5 (может уменьшиться через события, может увеличиться через реликвии) |
| Восстановление | Пропуск хода (+1 HP), события, реликвия «Фляга» |
| Между боями | HP **не восстанавливается** автоматически |

### Персонаж

| Параметр | Значение |
|----------|----------|
| Движение | 0–2 клетки за ход (горизонталь, вертикаль, L-форма) |
| Действие | 1 за ход (бросок / толчок / пропуск) |
| Размер | 1×1 клетка |
| Визуал | Составной SDF: круг (голова) + треугольник (плащ). Внутри — Flow noise цвета последнего использованного зелья. Idle: лёгкое покачивание |

### Смерть

HP ≤ 0 → ран окончен. Анимация смерти (dissolve + частицы). Переход на экран результатов:

- Глубина: этаж + номер узла
- Враги убиты: число
- Зелья использованы: иконки
- Combo выполнено: число
- Лучший combo: описание
- Прогресс ачивок (если продвинулся)
- Кнопка «Ещё раз» (мгновенный рестарт)
- Кнопка «Лаборатория» (главное меню)

### Победа

Убит финальный босс → Экран победы. Та же статистика + поздравление. Разблокировки мета-прогрессии показываются здесь.

## 11. Мета-прогрессия

### Принцип

**Никакой валюты.** Выполнил условие → разблокировал награду навсегда. Каждая ачивка учит механике, которую награда усиливает. Прогрессия сохраняется между ранами.

### Категория: Рецепты

В начале доступно **3 рецепта** и **3 базовых зелья**. Остальные открываются ачивками.

**Стартовые рецепты:** Пар (🔴+🔵), Напалм (🔴+🟢), Кислота (🔵+🟢)

**Стартовые базовые:** Пламя (🔴+🔴), Поток (🔵+🔵), Отрава (🟢+🟢)

| Разблокировка | Условие | Чему учит |
|---------------|---------|-----------|
| ✨ Сияние (🟡+🟡) | Дойди до Этажа 2 | Знакомство с новым элементом |
| 🌑 Мрак (🟣+🟣) | Дойди до Этажа 3 | Знакомство с новым элементом |
| ⚡ Молния (🔵+🟡) | Убей 3 врагов зельем воды за 1 ран | Поток → Молния (апгрейд стихии воды) |
| 💥 Вспышка (🔴+🟡) | Оглуши врага, стоящего в огне (combo Пламя+Вспышка) | Combo-мышление |
| 💀 Проклятие (🔴+🟣) | Убей элитного врага без получения урона | Контроль позиции |
| 🌊 Бездна (🔵+🟣) | Толкни 5 врагов суммарно (за все раны) | Толчок как механика |
| 🌿 Спора (🟢+🟡) | Используй событие «Фонтан» 3 раза | Исследование событий |
| 🕸 Миазма (🟢+🟣) | Нанеси ядом 30 урона суммарно (DoT считается) | Зоны и DoT |
| 🌀 Хаос (🟡+🟣) | Открой все остальные рецепты | Мастерство всех систем |

При разблокировке рецепта:
- Соответствующие элементы начинают появляться на доске перегонки (если ещё не появлялись)
- Зелье появляется в Книге алхимика (коллекция)
- Однократная визуальная нотификация с описанием

### Категория: Реликвии

Реликвии — пассивные предметы, действующие весь ран. Разблокированные реликвии попадают в **пул**. После победы над элитой игрок выбирает **1 из 3** случайных реликвий из пула. Максимум **3 реликвии** за ран.

| Реликвия | Эффект | Условие разблокировки |
|----------|--------|----------------------|
| 🫙 Термос | 25% шанс: кулдаун зелья не тратится при использовании | Используй 30 зелий суммарно |
| 🔍 Линза | AoE всех зелий +1 клетка | Задень 3 врагов одним зельем |
| 🧬 Мутаген | При merge на доске перегонки 20% шанс: одна случайная пустая клетка заполняется бонусным ингредиентом | Сделай 30 merge суммарно |
| ❤️ Фляга | +1 HP после каждого боя (не выше max) | Пройди 3 боя подряд без получения урона |
| 🎲 Кубик | +1 действие на фазе перегонки (4 вместо 3) | Создай зелье lv3 |
| 💎 Призма | Combo-эффекты наносят ×1.5 урона | Сделай 5 combo суммарно |
| 🐌 Декантер | Зоны на поле длятся +1 ход | Создай 3 зоны на поле за 1 бой |
| 🗡 Шип | Толчок наносит 2 урона | Убей врага толчком (в яму или в стену с добивающим уроном) |
| 🌀 Вихрь | В начале каждого боя 1 случайная зона появляется на случайной клетке поля | Пройди 5 комнат с модификаторами |
| 👁 Монокль | Видишь намерения врагов на **2 хода вперёд** (текущий + следующий) | Дойди до Этажа 3 |

### Категория: Постоянные улучшения

Необратимые бонусы, улучшающие стартовые условия.

| Улучшение | Эффект | Условие |
|-----------|--------|---------|
| 5-й слот для зелий | Инвентарь: 5 слотов вместо 4 | Победи финального босса |
| Расширенная доска 5×4 | 20 ячеек вместо 16 на перегонке | Используй все 5 элементов за 1 ран |
| 4-е действие | 4 действия за фазу перегонки (вместо 3) | Создай хотя бы 1 зелье каждого разблокированного рецепта (за все раны) |
| Стартовое зелье | Начинай ран с 1 случайным зельем lv1 | Дойди до Этажа 3 три раза |
| Дальнозоркость | Видишь 2 узла вперёд на карте (вместо 1) | Пройди 10 событий |

## 12. Обучение

### Принцип

Первый ран — обучающий. Минимум текста. Подсказки — стрелки, подсветка, вибрация (haptic). Прогрессивное раскрытие: каждый шаг вводит **одну** механику.

### Шаг 1: Merge

Доска содержит только 🔴 и 🔵 (упрощённая генерация). Количество: 10× 🔴 и 6× 🔵.

Подсказка: анимированная стрелка от 🔴 к соседней 🔴. Пульсирующая подсветка обеих капель.

Действие: Игрок перетаскивает 🔴 → 🔴. Merge → появляется варево [🔥 Пламя lv1] с burst-эффектом.

Подсказка: стрелка от 🔵 к 🔵. Игрок мержит → [🌊 Поток lv1].

Все 3 действия потрачены. Автоматический сбор варев → два зелья в инвентаре. Слоты зелий подсвечиваются.

**Длительность: ~30 сек.**

### Шаг 2: Первый бой

Комната 4×4 (уменьшенная для обучения). Один скелет в углу.

Подсказка: «Тапни зелье → тапни клетку». Подсвечивается зелье Пламя → подсвечиваются допустимые клетки.

Действие: Игрок бросает Пламя → скелет получает урон → умирает (dissolve). Победа.

**Длительность: ~15 сек.**

### Шаг 3: Движение и кулдаун

Новая фаза перегонки. Доска проще (8 🔴, 8 🔵). Игрок делает merge (или подсвечивается подсказка). Зелья собираются.

Бой: 5×5, два скелета, игрок в углу. Подсказка: свайп от персонажа (стрелка) → движение.

Игрок бросает зелье → оно тускнеет, число «2» (кулдаун). Подсказка: указывает на второе зелье. Игрок понимает чередование.

**Длительность: ~45 сек.**

### Шаг 4: Подпитка и рецепт

Фаза перегонки. На доске — 🔴, 🔵, 🟢.

**Часть A: Рецепт.** Подсказка: стрелка от 🔴 к соседней 🟢. Merge → появляется [☣️ Напалм lv1]. Книга рецептов (📖) мигает. Тап → открывается книга с одним рецептом.

**Часть B: Подпитка.** Подсказка: стрелка от 🔴 (соседней с варевом Напалм) к варево. Drag → Напалм lv1 → lv2. Визуальный акцент: варево увеличивается, glow усиливается.

Последнее действие — свободное (игрок делает что хочет).

**Длительность: ~30 сек.**

### Шаг 5: Свободная игра

Обучение завершено. Появляется карта этажа. Игрок выбирает путь. Никаких подсказок (кроме кнопки 📖 Книга, которая пульсирует первый раз).

**Суммарная длительность обучения: ~2 минуты.**

## 13. Визуальный стиль

### Название стиля: Liquid Geometry

Все объекты = чёткий SDF-силуэт + живое шейдерное наполнение. Ничего статичного — всё дышит, пульсирует, перетекает. Эстетика: алхимическая лаборатория в подземелье, увиденная через магические линзы.

### Палитра

| Слой | Цвет | Назначение |
|------|------|------------|
| Фон | #0A0A12 → #1A1A2E | Глубокий тёмно-синий, неподвижный |
| Стены | #12121F → #252540 | Чуть светлее фона, Voronoi noise |
| Пол | #1E1E32 → #2A2A45 | Voronoi noise (камни), швы слегка светятся |
| Объекты | Яркие, насыщенные | Каждый элемент = свой цвет (см. таблицу) |
| Эффекты | Яркие + bloom | Самые яркие объекты на экране |
| UI | #E0E0E0 (текст), #424242 (рамки) | Чистый, не конкурирует с игрой |

### Принцип: Фон ВСЕГДА тёмный, интерактивные объекты ВСЕГДА светятся.

### Иерархия яркости (от ярчайшего к темнейшему)

```
1. Эффекты зелий и combo (bloom, glow, частицы)
2. Зелья в инвентаре и варева на доске
3. Ингредиенты на доске перегонки
4. Персонаж игрока
5. Враги (силуэт + акцентный цвет)
6. Зоны на поле (полупрозрачный эффект)
7. Интерактивные объекты (столбы, котлы)
8. Тайлы пола (едва видимая текстура)
9. Стены (минимальная видимость)
10. Фон (почти чёрный)
```

### Ключевые шейдерные техники

| Объект | Техника | Детали |
|--------|---------|--------|
| Пол подземелья | Voronoi noise | Каменная кладка. Швы светятся цветом стихии модификатора комнаты (если есть). Intensity: 0.05–0.1 |
| Ингредиенты | SDF metaball + element-specific fill | Покачивание (sin на позиции, period 2 сек). При drag — деформация (stretch в направлении drag). При merge — притяжение (lerp к позиции цели) |
| Варево | SDF колба (прямоугольник + круг горлышко) | Жидкость внутри: sine wave на upper boundary. Пузырьки: circle SDF, float up. Glow edge: emission на контуре колбы. Уровень = высота заполнения (33% / 66% / 100%) |
| Персонаж | Составной SDF (круг + треугольник) | Внутри — Flow noise, цвет = последнее использованное зелье. Idle: покачивание (sin, period 3 сек). При движении: squash & stretch |
| Скелет | SDF прямоугольник + круг | Бледно-жёлтый. Мерцающие точки-глаза (sin pulse). Покачивание |
| Паук | SDF овал + 8 линий | Ноги: sin wave с offset per leg. Красные глаза. Body pulsation |
| Гриб-бомба | SDF полукруг + прямоугольник | Color lerp по таймеру. Пульсация с ускорением |
| Магнит-голем | SDF квадрат (скруглённый) | Voronoi fill. Концентрические ring SDF (притяжение) |
| Зеркальный слайм | SDF metaballs (3-4 круга) | HSV hue shift (2 сек цикл). Хроматическая аберрация при копировании |
| Бронированный жук | SDF шестиугольник | Specular highlight. Щит-индикатор: 2 ромба |
| Фантом | SDF круг + trail | Alpha: sin oscillation 0.3–0.8. Trail: 3 круга с убывающей alpha |
| Некромант | SDF треугольник + прямоугольник | Спиральный UV warp. Руны: scrolling step function |
| Огонь (эффект) | fBm noise | Маскированный AoE-формой. 3 октавы. Color ramp: orange→yellow→white |
| Молния | Jagged line | Сегментированная ломаная: 8 сегментов, random offset per frame (±0.15 клетки). Glow edge. Flash при hit |
| Кислота | Metaballs | Капли сливаются. 5-7 кругов с smooth union. Зелёный + жёлтый |
| Лёд | Voronoi cracks | Crack propagation от центра. White → light blue. Sharp edges |
| Яд | Warped circle | Perlin distortion на boundary. Всплывающие пузырьки (circle float up + fade) |
| Пар | Perlin noise облако | Low contrast, белый. Animated scroll. Semi-transparent |
| Dissolve (смерть) | Noise threshold | Grayscale noise → threshold mask. Emission edge (яркая полоса на границе dissolve). Duration: 0.4 сек |

### Пост-процессинг

| Эффект | Когда | Настройки |
|--------|-------|-----------|
| Bloom | Всегда | Threshold: 0.8, Intensity: 1.5, Radius: 4px |
| Виньетка | Всегда | Intensity: 0.3, Smoothness: 0.5 |
| Хроматическая аберрация | При сильном ударе | Intensity: 0.8, Duration: 0.1 сек, ease-out |
| Screen shake | При ударе, combo, смерти | Magnitude: 3-8px, Duration: 0.1–0.3 сек |
| Hitstop (slowmo) | При попадании зельем | Time.timeScale = 0.05, Duration: 0.05 сек |
| Fullscreen flash | При combo и crit | White overlay, alpha 0.4→0, Duration: 0.15 сек |

### Juice (список эффектов по событиям)

| Событие | Визуал | Камера | Время |
|---------|--------|--------|-------|
| Попадание по врагу | Белая вспышка на враге (flash shader). Числа урона подпрыгивают (bounce ease) | Shake (3px, 0.1s) | 0.15 сек |
| Убийство врага | Dissolve (0.4s). Частицы разлетаются (8-12 частиц, radial). Звёздочки опыта летят к игроку (bezier curve) | Shake (5px, 0.15s) | 0.5 сек |
| Combo | Расширенные визуальные эффекты (×2 particle count, ×1.5 size). Надпись «COMBO» (scale up + fade, gold color, glow) | Shake (8px, 0.25s) + Slowmo (0.15s) + Flash | 0.8 сек |
| Merge на доске | Притяжение ингредиентов (lerp 0.2s). Слияние (overlap + flash). Ring shockwave от точки merge (ring SDF expand + fade). Bounce нового варева (scale 0→1.2→1, ease-out-elastic) | — | 0.4 сек |
| Подпитка варева | Ингредиент летит к вареву (lerp 0.15s). Варево glow усиливается (intensity ×2 на 0.3s). Уровень: жидкость поднимается (0.2s) | — | 0.3 сек |
| Сбор варев в инвентарь | Варева взлетают с доски (bezier curve к слотам, 0.3s, stagger 0.1s). При приземлении — bounce + flash | — | 0.5 сек |
| Получение урона | Красная вспышка по краям экрана (red vignette, alpha 0.5→0, 0.3s). Персонаж: белая вспышка + knockback-анимация (0.1s) | Shake (5px, 0.15s) | 0.3 сек |
| Смерть игрока | Slowmo (0.5s). Dissolve персонажа (0.6s). Затемнение экрана (fade to black, 0.5s) | Shake (6px, 0.3s) | 1.5 сек |
| Появление врагов | Враги «проявляются» из темноты (alpha 0→1, 0.3s, stagger 0.15s) | — | 0.5 сек |

- **Доска**: сетка 4×4 с ингредиентами и варевами
- **Точки действий**: три точки вверху. Заполненная = потрачена. Анимация: гаснет при трате
- **Превью**: при наведении пальца на варево — показывает название, уровень, описание зелья
- **Кнопка «В бой»**: можно завершить фазу досрочно (если осталось <3 действий или игрок доволен результатом)

### Drag feedback на доске перегонки

| Состояние | Визуал |
|-----------|--------|
| Палец на ингредиенте | Ингредиент увеличивается (×1.2). Подсвечиваются валидные соседние клетки (зелёная рамка) |
| Drag к соседнему ингредиенту | Линия-связь между ними. Превью результирующего зелья появляется рядом (полупрозрачно) |
| Drag к соседнему вареву | Стрелка от ингредиента к вареву. Если допустимая подпитка — варево пульсирует и показывает «+1 lv». Если недопустимая — красный X |
| Drag к пустой клетке | Серый X. Действие не произойдёт |
| Отпускание на валидной цели | Merge/подпитка с анимацией |
| Отпускание вне цели | Ингредиент возвращается на место (bounce back) |

## 15. Звук

### Принцип

Процедурная генерация звуков (sfxr / jsfxr / подобные инструменты) для единообразия с визуальным стилем. Все звуки — синтетические, «алхимические».

### Звуковые категории

| Категория | Стиль | Примеры |
|-----------|-------|---------|
| Merge / перегонка | Жидкие, булькающие | Merge: «буль» + rising tone. Подпитка: «шипение» + chime |
| Боевые | Резкие, ударные | Удар: short noise burst. Огонь: crackling noise. Молния: sharp zap |
| UI | Чистые, мелодичные | Тап: soft click. Переход: whoosh. Открытие книги: page flip |
| Ambient | Тёмный, подземный | Low drone + occasional drip. Subtler per floor |

### Музыка

Рекомендация: минималистичный ambient loop (60–90 сек). Разный тон по этажам:
- Этаж 1: спокойный, загадочный
- Этаж 2: напряжённый, ритмичный
- Этаж 3: тревожный, давящий
- Босс: интенсивный, ударные

## 16. Технические спецификации

### Архитектура

| Компонент | Подход |
|-----------|--------|
| Game state | ScriptableObject-based state machine (Menu → Distillation → Combat → Map → ...) |
| Board logic | Pure C# model (no MonoBehaviour). View отдельно |
| Combat | Turn-based state machine. Command pattern для undo |
| Enemy AI | Strategy pattern (каждый тип врага = IEnemyBehavior) |
| Save | JSON serialization. PlayerPrefs для мета-прогрессии, файл для mid-run save |
| Shaders | Shader Graph (URP-compatible SDF nodes) или hand-written HLSL |
| UI | Unity UI Toolkit (или Canvas + anchors) |

### Производительность

| Метрика | Цель |
|---------|------|
| FPS | 60 стабильных |
| Memory | <200 MB |
| APK size | <50 MB |
| Battery | <10% за 30 мин сессии |
| Load time | <3 сек от тапа до меню |

### Целевые устройства

| Минимум | Рекомендуемое |
|---------|---------------|
| Android 8.0 (API 26) | Android 12+ |
| 2 GB RAM | 4 GB RAM |
| Mali-T720 / Adreno 306 | Mali-G76 / Adreno 618+ |
| 720×1280 | 1080×1920 |

### Оптимизация шейдеров

- Все SDF-объекты — один material per batch (GPU instancing)
- Noise-функции: предрасчитанные в текстуру (256×256 tileable) для слабых GPU. Fallback: процедурный на сильных
- Bloom: dual-pass Kawase blur (дешевле Gaussian)
- Particle-like эффекты: шейдерные (не Unity Particle System). Mesh quad + UV animation
- Батчинг: все враги одного типа — один draw call. Все ингредиенты — один draw call
